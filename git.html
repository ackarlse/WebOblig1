GITHUB

Git er et verktøy man bruker for å "commite" endringer man har gjort til et kode dokument til Github. Man kan "commite" nye dokumenter, bilder, tekst etc.. altså hele prosjektet man jobber med. Git gir deg og andre full versjonkontroll, som vil si at man kan gå tilbake til tidligere i prosjektet. Man kan også bruke Git til å dele kode, som finnes på Github kontoen. Github derimot er tjenesten man bruker til å lagre alt, det finnes også flere andre slike tjenester, som: BitBucket eller GitLab.
